_ Những điều em đã làm:
   1) User launches the app. An icon  will be added to notification area
   2) View notes by tags
   3) User presses the shortcut key (like ​Windows + Space​), a dialog will appear
   4) User types the note, enter multiple tags, seperated by commas (“,”) then click Save
   5) After a while of using our app, user can see the statistics chart based on his usage frequency.
Kịch bản
1. Thêm Tag mới:
_ Kịch bản chính (Thành công):
   + Nhấn tổ hợp phím Ctrl + Space để mở Dialog thêm note
   + Nhập tag: Ăn uống
   + Nhập Content: đi ăn kem với bạn ngày 20/01/2017
   + Chọn mục TagSugget sau đố nhấn Add Tag nếu muốn sử dụng Tag gợi ý
   + Nhấn Save để lưu note lại
_ Kịch bản phụ (thất bại) thứ nhất:
   + Để trống mục Tag
   + Bấm Save, báo lỗi "không để trống Tag"
_ Kịch bản phụ (thất bại) thứ hai:
   + Để trống mục Content
   + Bấm Save, báo lỗi "không để trống Content"
_ Kịch bản phụ (thất bại) thứ ba: (khi không có dữ liệu trong CSDL nghĩa là sẽ không có tag gợi ý)
   + bấm Add Tag
   + báo lỗi "không có Tag gợi ý"
2. View Notes
_ Kịch bản chính (thành công)
   + Bấm chuột phải vô notify icon chọn "View Notes", xuất hiện Dialog View Notes
   + Nhấp đúp chuột trái vô một dòng để xem thông tin Tag, xuất hiện Dialog với list view 1 cột là tag 1
     cột là content
   + Nhấp đúp chuột trái vô một dòng để xem thông tin chi tiết, xuất hiện Dialog với 2 textbox,1 là Tag
     còn lại là content.
_ Kịch bản phụ (thất bại): Chưa tìm ra
3. View Statistics:
_ Kịch bản chính (thành công)
   + Bấm chuột phải vô notify icon chọn "View Statistics", xuất hiện Dialog View Statistic với biểu đồ tròn và các
     ghi chú tương ứng
_ Kịch bản phụ (thất bại): Chưa tìm ra
